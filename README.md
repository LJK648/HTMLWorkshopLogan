Vernball – Makeshift Sports Webpage

Overview

Vernball is a lightweight, makeshift sports webpage project built to demonstrate core web development concepts in a sports-themed context.
The site focuses on structure, layout, and basic interactivity rather than production-level polish. Think proof-of-concept, not ESPN.

The goal is simple: present sports-related content (teams, scores, stats, or news) in a clean, readable format while practicing front-end fundamentals.

Purpose

This project was created to:
	•	Practice HTML, CSS, and basic JavaScript
	•	Experiment with page layout and styling
	•	Simulate a real-world sports webpage without backend complexity
	•	Serve as a foundation that could be expanded later
